# Security Policy

## Scope

This repository contains **documentation and configuration templates** for a self-hosted OpenClaw deployment. It does not contain executable application code.

No secrets, credentials, or private keys are stored here. All sensitive values referenced in config templates are explicitly redacted.

## Reporting a Vulnerability

If you find a security issue in this repo (e.g. accidentally committed credentials, an unsafe config pattern), please **do not open a public issue**.

Contact me directly:

- **Email**: ayetek0773@gmail.com

I will respond within 48 hours and address confirmed issues promptly.

## For OpenClaw Vulnerabilities

If the issue is in OpenClaw itself (not this repo), please report it to the OpenClaw project directly:

- [openclaw/openclaw](https://github.com/openclaw/openclaw)
